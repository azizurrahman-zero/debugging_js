JavaScript is a single-threaded, interpreted scripting language. In JavaScript, the code is interpreted by the browsers. To run JavaScript every browser has a dedicated JavaScript engine (Chrome has V8).
The engines take source code line by line and firstly check syntactic errors by 'Parser'. If 'Parser' finds any error there it stops the execution else it produces a data structure named abstract syntax tree(AST) which translated the code to machine code. Then the code is sent to the system for execution and after execution, the system runs the code and displays output.

Local Storage is that kind of Web Storage that stores data for a long time. On the other hand, Session Storage stores data for a short time(session). Though there are many similarities between them (Such as Only Client-Site accessibility, Never transfer data to the server), there are some differences too. The differences are:
Basis Local Storage Session Storage
Expiration Never Expired Expired when Browser Tab is closed
Storage Capacity About 10 MB About 5 MB
Accessible Data from Any window of Browser Same Tab of Browser
